# Gyro-Mouse
A flutter project that allows the user to control the mouse in linux using xdotool 
# Prerequisite
##install xdotool 
on debian
````
sudo apt install xdotool
````
on fedora
````
sudo dnf -y install xdotool
````
on Arch
````
pacman -S xdotool
````
then run the server using 
````
dart server.dart
````
